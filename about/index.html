Implementation of key features in JOS kernel

Sept. 2018 - Nov. 2018

JOS is a toy exokernel-like operating system developed by MIT. I will implement some key features in the kernel.

Here are what I have implemented:

  • booting and memory management part.
  • add support for 4MB big page in memory management
  • monitor instructions for showing memory mappings in a range of address
  • running programs in user environment and trap/interrupt handling
  • intergrated intel fast system call instructions(sysenter/sysexit), improved performance by 2x than using int/iret system call
  • Supported running on a multiprocessor system and implemented Inter-Process Communication with shared memory
  • Built a UNIX like on-disk file system to support running a shell on console
  • Wrote network drivers for E1000 network interface card, which supports running a web server on the top of JOS

Optimize the performance of HPCG Benchmark on cluster

Sep 2015 – Jan 2016

The High Performance Conjugate Gradients (HPCG) Benchmark is intended as a complement to the High Performance LINPACK (HPL) benchmark, currently used to rank the TOP 500 computing systems. Our job is optimizing it’s performance on a 10 nodes cluster. We used MPI, OpenMP to optimize HPCG, including computational schemes, compiling, memory accessing, communication. And analysis performance bottlenecks with Vtune, Paramon, etc. Finally we achieved a performance enhance by 10.5 GPLOP/s.
